# Project Overview

## Goal
Create a personal portfolio website for Heron Chavez that showcases Support Operations experience, Automation projects, and AI-driven workflows. The site focuses on real projects with Loom demos and GitHub links.

## Users
- **Hiring managers / recruiters** — evaluating Heron for technical roles
- **Potential clients** — looking for automation and support operations services

## Core Features
- **Home page** — Hero section with name/title, about section, featured projects
- **Projects page** — Grid of project cards with demo/code/details links
- **Project detail pages** — Loom video embeds, problem/solution sections, tech stack, GitHub links
- **Navbar** — Home, Projects, Contact links
- **Dark theme** — Minimal, professional, responsive design

## Constraints
- No authentication
- No CMS
- No unnecessary animations
- Keep it simple and professional
- Production-ready and deployable to Vercel

## Definition of Done
- The change solves the requested problem.
- The affected behavior is validated.
- The change is limited to the requested scope.
- Any lasting project knowledge is added to docs or .ai files.
