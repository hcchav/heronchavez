# Commands

## Setup
```bash
npm install
```

## Run
```bash
npm run dev
```
Starts dev server at http://localhost:3000

## Test
No test framework configured yet.

## Lint
```bash
npm run lint
```

## Build
```bash
npm run build
```

## Deploy
Push to GitHub and connect to Vercel, or use Vercel CLI:
```bash
vercel
```

## Notes
- No environment variables required for basic functionality.
- Profile image should be placed at `/public/profile.jpg`.
